Quick Start Guide

1 Prerequisites

Ensure you have the .NET 8 SDK installed on both machines.

2 Setup the Receiver (PC)

Run the receiver with a pre-shared key (PSK). This key will be used to sign all input messages. 

dotnet run --project src/Receiver -- --psk your_secret_key

3 Setup the Sender (Laptop)

Run the sender with the same key and the IP address of your Receiver PC. 

dotnet run --project src/Sender -- --psk your_secret_key --receiver 192.168.1.50

Tip: Mode Toggling

Use Ctrl + Alt + F12 to toggle between "Locked" mode (PC control) and "Local" mode (Laptop control).
